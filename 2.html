CSS란 무엇인가?

CSS 는 레이아웃, 색상 및 글꼴을 포함하여 프레젠테이션과 콘텐츠를 분리 할 수 ​​있도록 설계되었습니다. 이러한 분리는 컨텐츠 접근성을 향상시키고, 프리젠테이션 특성의 사양에서 더 많은 유연성과 제어를 제공 하며, 구조적 컨텐츠의 복잡성과 반복을 줄여주는 별도의 .css 파일에 관련 CSS를 지정하여 여러 웹 페이지 에서 서식을 공유 할 수 있도록 합니다.

.css 파일을 캐시 할 수 있도록 설정하여 파일을 공유하는 페이지와 해당 형식 사이의 페이지로드 속도를 향상시킵니다. 서식과 내용을 분리하면 화면, 인쇄물, 음성 (음성 기반 브라우저 또는 화면 판독기 사용 ) 및 점자 기반 과 같은 다양한 렌더링 방법에 대해 서로 다른 스타일로 동일한 마크 업 페이지를 표시 할 수 있습니다. 촉각 장치. CSS에는 모바일 장치 에서 콘텐츠에 액세스하는 경우 대체 형식에 대한 규칙도 있습니다 .
