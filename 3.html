JavaScript란 무엇인가?

JavaScript는 HTML 및 CSS 와 함께 JavaScript는 World Wide Web 의 핵심 기술 중 하나입니다. JavaScript는 대화형 웹 페이지를 가능하게 하며 웹 응용 프로그램의 필수 부분입니다. 대부분의 웹 사이트는 이를 클라이언트 측 페이지 동작에 사용하고 모든 주요 웹 브라우저에는 이를 실행하기 위한 전용 JavaScript 엔진 이 있습니다.

다중 패러다임 언어 인 JavaScript는 이벤트 기반 , 기능적 , 명령형 프로그래밍 스타일을 지원 합니다. 여기에는 텍스트, 날짜, 정규식 , 표준 데이터 구조 및 DOM ( 문서 개체 모델 ) 작업을위한 API ( 응용 프로그래밍 인터페이스 )가 있습니다. 그러나 호스트 환경 (일반적으로 웹 브라우저)에서 해당 API를 제공 하므로 언어 ​​자체에는 네트워킹 , 스토리지 또는 그래픽 기능 과 같은 I / O ( 입 / 출력)가 포함되지 않습니다.
